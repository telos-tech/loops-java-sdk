Loops Java SDK

A type-safe Java SDK for the Loops.so API, providing full API coverage for contacts, events, transactional emails, mailing lists, and custom properties.

Features

  • Full API Coverage - Supports all Loops.so API endpoints
  • Async Support - Both synchronous and asynchronous (CompletableFuture) methods
  • Type-Safe - Leverages Java records for immutable, type-safe data models
  • Resilient - Built-in retry logic for rate limiting
  • Modern Java - Built for Java 21+

Quick Start


// Initialize the client
LoopsClient client = LoopsClient.builder()
        .apiKey("your-api-key")
        .build();

// Create a contact
ContactResponse response = client.contacts().create(
    ContactCreateRequest.builder()
        .email("user@example.com")
        .firstName("John")
        .lastName("Doe")
        .build()
);

// Send an event
EventResponse eventResponse = client.events().send(
    EventRequest.builder()
        .email("user@example.com")
        .eventName("signed_up")
        .build()
);

// Send a transactional email
TransactionalResponse emailResponse = client.transactional().send(
    TransactionalRequest.builder()
        .transactionalId("welcome_email")
        .email("user@example.com")
        .build()
);

Getting Started

To get started with the Loops Java SDK:

  1. Add the dependency to your project (Maven or Gradle)
  2. Obtain an API key from your Loops.so account
  3. Initialize the {@link com.telos.loops.LoopsClient} with your API key
  4. Use the various client methods to interact with the Loops.so API

Main Components

Core Client

  • {@link com.telos.loops.LoopsClient} - Main entry point for the SDK

API Clients

  • {@link com.telos.loops.contacts.ContactsClient} - Manage contacts (create, update, delete, find)
  • {@link com.telos.loops.events.EventsClient} - Send events to Loops
  • {@link com.telos.loops.transactional.TransactionalClient} - Send transactional emails
  • {@link com.telos.loops.lists.MailingListsClient} - Manage mailing lists
  • {@link com.telos.loops.properties.ContactPropertiesClient} - Manage custom contact properties
  • {@link com.telos.loops.ips.DedicatedIpsClient} - Manage dedicated IPs
  • {@link com.telos.loops.apikey.ApiKeyClient} - Retrieve API key information

Error Handling

  • {@link com.telos.loops.error.LoopsApiException} - Base exception for API errors
  • {@link com.telos.loops.error.LoopsValidationException} - Validation errors
  • {@link com.telos.loops.error.RateLimitExceededException} - Rate limit errors

Asynchronous Operations

All API methods have asynchronous variants that return {@link java.util.concurrent.CompletableFuture}. Asynchronous method names end with "Async": 


// Synchronous
ContactResponse response = client.contacts().create(request);

// Asynchronous
CompletableFuture<ContactResponse> futureResponse =
    client.contacts().createAsync(request);

// Handle the result
futureResponse.thenAccept(response -> {
    System.out.println("Contact created: " + response.id());
});

Requirements

  • Java 21 or higher
  • Maven 3.8+ or Gradle 7.0+
