How to Use These Examples

1. Set Your API Key

All examples read the API key from the LOOPS_API_KEY environment variable:

# Linux/macOS
export LOOPS_API_KEY=your-api-key-here

# Windows (Command Prompt)
set LOOPS_API_KEY=your-api-key-here

# Windows (PowerShell)
$env:LOOPS_API_KEY = "your-api-key-here"

2. Add the SDK Dependency

Ensure you have the Loops Java SDK in your project:


<dependency>
    <groupId>com.telos</groupId>
    <artifactId>loops-java-sdk</artifactId>
    <version>1.0.0</version>
</dependency>

// Gradle
implementation 'com.telos:loops-java-sdk:1.0.0'

3. Copy and Run

Copy the example code into your project and run it. Each example is self-contained with detailed comments explaining each step.

4. Customize

Modify the examples to fit your specific use case. Replace hardcoded values with your own data, add error handling, or integrate with your application logic.
