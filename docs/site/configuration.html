Configuration

Customize the Loops Java SDK client to fit your application's needs.

Base URL

By default, the SDK uses https://app.loops.so/api/v1 as the base URL. If you need to use a proxy or a different environment, you can configure it when building the client.

LoopsClient client = LoopsClient.builder()
    .apiKey("your-api-key")
    .baseUrl("https://your-proxy-url.com/api/v1")
    .build();

Timeouts

You can configure connection and read timeouts to ensure your application remains responsive even if the network is slow.

LoopsClient client = LoopsClient.builder()
    .apiKey("your-api-key")
    .connectTimeout(Duration.ofSeconds(5))
    .readTimeout(Duration.ofSeconds(30))
    .build();
